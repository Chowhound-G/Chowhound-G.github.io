# Gabriel Profile

Standalone personal portfolio for Gabriel.

## Stack

- React
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Content

- Profile, contact, and project data: `src/content/profile.ts`
- Markdown articles: `src/content/articles`
- Resume PDF: `public/Gabriel-Resume.pdf`
