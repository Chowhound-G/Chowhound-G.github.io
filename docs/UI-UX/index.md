#### 27 裁剪图片的七大法则

1. 硬边裁剪

2. 柔和裁剪(羽化):轻盈 通透 略带模糊 [图像无限延展的感觉]
3. 分隔裁剪: 文字可以放在裁剪位置
4. 延伸裁剪: 营造遮挡的效果
5. 分离裁剪/剪影: 去除所有背景
6. 虚假裁剪: 裁剪部分,之后又露出[用于封面、宣传册]
7. 形状裁剪: CD 

可以结合使用

#### 28 人眼和思想意识的区别

#### 29 中心、边缘和三分法则

中心: 平稳休息、冷静

边缘: 制造混乱 带来无限的遐想和不安感

构图: 三分法则 九宫格用于交叉点
